# Contexts

Esta pasta concentra os contextos funcionais e arquiteturais do projeto.

Arquivos:
- `backend_context.md`: contexto atual do coletor e da camada de dados
- `frontend_context.md`: contexto da camada desktop e experiencia do usuario
- `app_scope.md`: escopo funcional aprovado da primeira versao do app

Objetivo:
- manter backend e frontend documentados separadamente
- facilitar evolucao futura sem misturar responsabilidades
